# Legend

## legend example and sandbox here

```jsx live
function Clock(props) {
  const [date, setDate] = useState(new Date());
  useEffect(() => {
    var timerID = setInterval(() => tick(), 1000);

    return function cleanup() {
      clearInterval(timerID);
    };
  });

  function tick() {
    setDate(new Date());
  }

  return (
    <div>
      <h2>It is {date.toLocaleTimeString()}.</h2>
    </div>
  );
}
```


Each chart has the option to include a legend which shows what category from your data each color represents. The legend and chart are both responsive, so they will resize to make space for each other.

Enable the legend by passing a value into the *legend* attribute. The following attributes are used to customize the appearance of the legend:

## Props

### legend

| Prop     | Type               | Optional | Default |
| -------- | ------------------ | -------- | ------- |
| `legend` | `boolean` or `"top"`, `"bottom"`, `"left"`, `"right"` , `"top-left"`, `"top-right"`, `"bottom-left"`, `"bottom-right"`, `"left-bottom"`, `"right-bottom"`, `"left-top"`, `"right-top"` | `true`   | `false`  |

*If `true`, the legend will default to position `right`.*

### legendLabel

This is the title at the top of the legend. 
*If the value is an empty string (`''`), the legend label will not render.*

| Prop    | Type           | Optional | Default |
| ------- | -------------- | -------- | ------- |
| `legendLabel` | `string` | `true`   | `''`    |
